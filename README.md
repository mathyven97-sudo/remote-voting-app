# Remote Voting App

## 📌 Overview
This is a mobile-based polling application that allows users to vote remotely. It is designed especially for people who cannot attend voting physically.

## 👥 Users
- Admin: Create polls and view results
- User: Vote on polls

## 🚀 Features
- Admin Login
- User Login
- Create Poll
- Vote
- View Results
- Role-based access control

## ⚙️ Technologies Used
- HTML
- CSS
- JavaScript
- Android WebView
- LocalStorage

## 🔄 Workflow
1. Admin creates a poll
2. Users vote
3. Results are displayed instantly

## 💡 Key Highlights
- Lightweight application
- No backend required
- Fast and efficient using local storage

## 📱 Future Improvements
- Add database (MySQL/Firebase)
- Add authentication system
- Add charts for results
